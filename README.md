💬 TCP Chat (C# / .NET / Socket Programming)

Сервер-клиентное приложение для обмена сообщениями по TCP-протоколу с поддержкой многопользовательского взаимодействия.

📌 Описание проекта
 
Данное приложение представляет собой чат, реализованный с использованием TCP-сокетов.

Система состоит из сервера и клиентов, которые могут подключаться и обмениваться сообщениями в реальном времени.

Приложение позволяет:
Подключать несколько клиентов к серверу
Отправлять и получать сообщения
Обрабатывать подключения и отключения пользователей
Работать через TCP-протокол
🛠 Используемые технологии
C# (.NET)
TCP Sockets
Асинхронное программирование (async/await / threads)
Клиент-серверная архитектура

🏗 Архитектура проекта

Проект разделён на две основные части:

Server — обрабатывает подключения клиентов и пересылает сообщения
Client — подключается к серверу и отправляет/получает сообщения

Основной принцип работы:

Сервер слушает порт
Клиенты подключаются
Каждое подключение обрабатывается отдельно
Сообщения рассылаются другим клиентам

🚀 Реализованные функции

✔ Подключение клиентов к серверу
✔ Отправка сообщений

✔ Поддержка нескольких клиентов
✔ Обработка отключений

🌐 Сетевая модель

Приложение работает по модели:

Client → Server → Clients

Клиент отправляет сообщение серверу
Сервер принимает и обрабатывает его
Сервер рассылает сообщение другим подключённым клиентам

⚙ Запуск проекта
🔹 1. Запуск сервера

Сначала необходимо запустить сервер:

dotnet run --project Server

или через IDE (Visual Studio / Rider)

🔹 2. Запуск клиентов

После запуска сервера запускаются клиенты:

dotnet run --project Client

Можно запустить несколько клиентов одновременно.

🔹 3. Настройка подключения

Убедитесь, что:

IP-адрес сервера указан корректно
Порт совпадает (например: 127.0.0.1:5000)

Клиенты получают и отображают сообщение
